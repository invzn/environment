# Neovim Configuration

## LSP Setup
- lua-language-server
```
brew install lua-language-server
```
- ccls
```
brew install ccls
```
- elixir-ls
```
brew install elixir-ls
```
- gopls
```
go install golang.org/x/tools/gopls@latest
```
- zls
```
brew install zls
```
- typscript-language-server
```
npm install -g typescript-language-server typescript
```
